Create a profile

To create a profile, use the Addressables Profiles window.


The Addressables Profiles window displaying the default profile.

To create a new profile:

  1. Open the Addressables Profiles window (Window > Asset Management > Addressables > Profiles).
  2. Select Create > Profile.

A new profile row appears in the table. Every profile must define a value for every variable. When you create a new profile, Addressables copies all values from the selected profile.

Set the active profile

The active profile determines the set of variables used when you run a build script.

To set the active profile, either:

  1. Open the Groups window (menu: Window > Asset Management > Addressables > Groups).
  2. Select the Profile menu in the toolbar.
  3. Choose the profile to make active.

Or:

  1. Open the Profiles window (menu: Window > Asset Management > Addressables > Profiles).
  2. Right-click on a profile to open the context menu.
  3. Choose Set Active.
